# Mood Light Server

## Description

Server that supports REST API for the **Mood Light** mobile app.

<br/>

## Installation

```bash
$ npm install
```

<br/>

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

<br/>

## Admin app

```bash
# development
$ npm run start:admin:dev

# production mode
$ npm run start:admin:prod
```

<br/>
